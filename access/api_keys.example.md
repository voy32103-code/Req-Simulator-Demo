# API Keys Example

Do not commit real API keys.

```text
OPENAI_API_KEY=replace-with-real-key
GEMINI_API_KEY=replace-with-real-key
AZURE_OPENAI_ENDPOINT=replace-with-real-endpoint
AZURE_OPENAI_API_KEY=replace-with-real-key
```

For the static MVP prototype, no external AI key is required. The stakeholder behavior is simulated locally with rule-based hidden requirement gating.
